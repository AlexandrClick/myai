# 🚀 Antigravity & Gemini MCP Build

Оптимизированное окружение для AI-кодинга (Gemini, Antigravity), использующее мощь графовых MCP-серверов для глубокого понимания кодовой базы.

Этот билд превращает обычного AI-ассистента в продвинутого инженера, который видит проект не как набор текстовых файлов, а как живой граф связей, процессов и зависимостей.

---

## 🛠️ Три основных компонента

Билд опирается на три мощных MCP-сервера:

1.  **[GitNexus](https://github.com/abhigyanpatwari/GitNexus) (Code Intelligence)**:
    *   Анализ цепочек вызовов и бизнес-процессов.
    *   Проверка "радиуса поражения" (`impact analysis`) перед правками.
    *   Безопасное переименование символов по всему проекту.
2.  **[CodeGraph](https://github.com/SuatKocar/codegraph) (Fast Navigation)**:
    *   Мгновенный поиск символов (классов, функций, типов).
    *   Быстрая навигация по структуре файлов и графу зависимостей.
    *   Легкое извлечение контекста для конкретной задачи.
3.  **[Code-Review-Graph](https://github.com/tirth8205/code-review-graph) (Structural Intelligence)**:
    *   Умные код-ревью с учетом изменений в архитектуре.
    *   Токен-эффективное извлечение только нужных кучков кода.
    *   Анализ покрытия тестами и поиск зависимых модулей.

## ⚙️ Установка и инициализация

Для работы некоторых компонентов рекомендуется использовать [uv](https://github.com/astral-sh/uv).

### 0. Установка UV
Если у вас еще не установлен `uv`, выполните:
```bash
curl -Ls https://astral.sh/uv/install.sh | sh
```

### 1. Инициализация инструментов
Запустите установку и индексацию для каждого компонента:

```bash
# 1. GitNexus (через npx)
npx gitnexus analyze

# 2. CodeGraph
codegraph init -i

# 3. Code-Review-Graph (рекомендуется через uv)
uv tool install code-review-graph
code-review-graph build
```

### 2. Очистка и настройка структуры
Чтобы правила применялись глобально из `~/.gemini/` и не засоряли проект:

1. Удалите локальные файлы `CLAUDE.md` и `AGENTS.md` (если они были созданы).
2. Переименуйте папку `.claude/` в `.agent/` (стандарт для Antigravity).
3. Добавьте служебные директории в `.gitignore`:

```text
.gitnexus
.codegraph
.code-review-graph/
.agent
```

## 📖 Как это использовать

После установки AI-ассистент будет автоматически использовать эти инструменты. Вам не нужно вручную вызывать команды, но полезно знать основные правила билда:

*   **Сначала граф, потом текст**: Ассистент сначала запрашивает структуру через MCP и только потом читает файлы. Это экономит токены и время.
*   **Анализ влияния (Impact)**: Перед любой правкой функции или класса ассистент обязан проверить, что и где может сломаться.
*   **Безопасный рефакторинг**: Переименования и перемещения кода делаются через умные инструменты, а не простым поиском и заменой.

### Полезные команды для обслуживания
Если индекс устарел (например, после крупных правок), можно его обновить:

*   `npx gitnexus status` / `npx gitnexus analyze`
*   `codegraph status` / `codegraph sync`
*   `code-review-graph status` / `code-review-graph update`
